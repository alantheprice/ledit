Installation

Automated Installation

The easiest way to install ledit is by using our platform-specific install scripts:

Linux & macOS

curl -fsSL https://raw.githubusercontent.com/ledit-dev/ledit/main/install.sh | bash

This script will download the latest ledit binary and install it to /usr/local/bin. You may be prompted for your password to use sudo.

Windows (PowerShell)

irm https://raw.githubusercontent.com/ledit-dev/ledit/main/install.ps1 | iex

This script will download the latest ledit binary and install it to your user's local application data directory, then add it to your user's PATH. You may need to restart your terminal or PowerShell session for the changes to take effect.

Manual Installation (From Source)

git clone https://github.com/ledit-dev/ledit.git
cd ledit
go build -o ledit .
# Move the 'ledit' executable to a directory in your PATH, e.g., /usr/local/bin or C:\Windows

Quick Start

# Initialize project
ledit init

# Create new file
ledit code "Python factorial function" --filename math.py

# Modify existing code
ledit code "Add error handling" --filename math.py

# Complex feature orchestration
ledit orchestrate "Implement user authentication"

# Fix a bug or issue
ledit fix "Fix the null pointer dereference in main.go" --filename main.go

# View change history
ledit log

# Ignore files or security concerns
ledit ignore add .env --type file
ledit ignore add "Insecure deserialization" --type security

# Ask a question about your code
ledit question "What does the `ProcessCodeGeneration` function do?" --filename pkg/editor/editor.go
